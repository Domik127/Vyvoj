# Postup pro vývoj a export na GitHub

1. **Inicializace repozitáře**
    ```bash
    git init
    ```

2. **Vytvoření souborů a úpravy**
    - Přidejte a upravte potřebné soubory projektu.

3. **Přidání změn do indexu**
    ```bash
    git add .
    ```

4. **Commit změn**
    ```bash
    git commit -m "První commit"
    ```

5. **Vytvoření repozitáře na GitHubu**
    - Na [github.com](https://github.com) vytvořte nový repozitář.

6. **Přidání vzdáleného repozitáře**
    ```bash
    git remote add origin https://github.com/uzivatel/jmeno-repozitare.git
    ```

7. **Odeslání změn na GitHub**
    ```bash
    git push -u origin master
    ```

8. **Další vývoj**
    - Opakujte kroky 2–7 pro další změny.
